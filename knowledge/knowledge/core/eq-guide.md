# EQ Frequency Guide

## Frequency Spectrum Overview

| Range | Frequency | Character | Common Uses |
|-------|-----------|-----------|-------------|
| Sub-bass | 20-60 Hz | Felt, not heard | 808, sub bass, kick weight |
| Bass | 60-250 Hz | Warmth, body | Bass guitar, kick punch, low vocals |
| Low-mids | 250-500 Hz | Muddy zone | Often cut to clean mix |
| Midrange | 500 Hz - 2 kHz | Body, presence | Vocals, guitars, snare |
| Upper-mids | 2-4 kHz | Presence, aggression | Vocal clarity, guitar bite |
| Presence | 4-6 kHz | Definition, edge | Vocal air, snare crack |
| Brilliance | 6-10 kHz | Sizzle, detail | Hi-hats, cymbals, consonants |
| Air | 10-20 kHz | Sparkle, openness | Overall mix sheen |

## Instrument-Specific EQ

### Vocals

| Frequency | Effect | Typical Action |
|-----------|--------|----------------|
| 80-120 Hz | Rumble, plosives | HPF cut |
| 200-300 Hz | Muddiness | Cut 2-4 dB |
| 400-600 Hz | Boxiness | Cut 2-3 dB if needed |
| 1-2 kHz | Nasal quality | Cut if honky |
| 3-5 kHz | Presence, clarity | Boost 2-4 dB |
| 6-8 kHz | Sibilance | DeEss or cut |
| 10-12 kHz | Air, breathiness | Gentle boost 1-3 dB |

### Electric Guitar

| Frequency | Effect | Typical Action |
|-----------|--------|----------------|
| Below 80 Hz | Unnecessary rumble | HPF cut |
| 100-200 Hz | Body, weight | Boost for thickness |
| 300-500 Hz | Mud, boxiness | Cut 2-4 dB |
| 800 Hz - 1.5 kHz | Honkiness | Cut if nasal |
| 2-4 kHz | Bite, attack | Boost for aggression |
| 5-8 kHz | Presence, pick attack | Boost for clarity |
| 10+ kHz | Fizz | Cut or leave natural |

| 600-800 Hz | Growl | Boost for presence |

## EQ Types and When to Use

### Parametric EQ
- **Use for**: Surgical cuts, precise frequency targeting
- **Best for**: Problem frequencies, narrow boosts

### Shelving EQ
- **Use for**: Broad tonal shaping
- **Best for**: Adding air (high shelf), warmth (low shelf)

### High-Pass Filter (HPF)
- **Use for**: Removing unwanted low frequencies
- **Best for**: Every track except kick and bass

### Low-Pass Filter (LPF)
- **Use for**: Removing harsh highs, creating distance
- **Best for**: Background elements, vintage sounds

### Dynamic EQ
- **Use for**: Frequency-dependent compression
- **Best for**: Taming resonances, controlling sibilance

## Pro Tips

1. **Cut narrow, boost wide** - Surgical cuts, gentle boosts
2. **Use HPF on everything** - Clean up the low end
3. **Reference constantly** - Compare to professional mixes
4. **Less is more** - Subtle moves add up
5. **Context matters** - Solo for problems, mix for balance
6. **Trust your ears** - These are guidelines, not rules

## Common Problem Frequencies

| Problem | Frequency | Solution |
|---------|-----------|----------|
| Mud | 200-400 Hz | Cut |
| Boxiness | 300-600 Hz | Cut |
| Nasal/honky | 800 Hz - 1.5 kHz | Cut |
| Harshness | 2-4 kHz | Cut or dynamic EQ |
| Sibilance | 5-8 kHz | DeEss or cut |
| Fizz/brittle | 8-12 kHz | Cut |
