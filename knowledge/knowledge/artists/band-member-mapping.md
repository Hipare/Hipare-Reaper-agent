# Band to Member Mapping

## Purpose

This file maps band names to their key members. When a user requests a "band sound", the AI should look up the individual members' sounds and combine them appropriately.

## How to Use

1. User says: "Create Meshuggah guitar tone"
2. AI looks up: Meshuggah → Guitarists: Fredrik Thordendal, Mårten Hagström
3. AI references: guitarists.md for their specific sounds
4. AI creates the combined sound

## Finnish Terms

- "bändi" = band
- "kitaristi" = guitarist
- "basisti" = bassist
- "rumpali" = drummer
- "kosketinsoittaja" = keyboardist
- "laulaja" = vocalist
- "soundi" = sound/tone
