# Band Production Styles - Complete Sound Guide

## Overview

This guide covers complete production approaches for famous bands. When a user requests a "band sound", combine these production techniques with individual member sounds from other files.

## Meshuggah

### Overall Production Style
- Surgically precise
- Extremely tight low-end
- Dry, controlled sound
- Polyrhythmic complexity

### Key Members
- Guitars: Fredrik Thordendal, Mårten Hagström → See guitarists.md
- Drums: Tomas Haake → See drummers.md
- Vocals: Jens Kidman → See vocalists.md

### Production Approach
```
DRUMS:
- Samples/triggers blended with acoustic
- Extremely tight gating
- Click-forward kick
- Dry, punchy snare

GUITARS:
- 8-string, F standard tuning
- Quad tracked (2L + 2R)
- Tight tube screamer before amp
- Modern high-gain tone

BASS:
- Follows guitar rhythm
- Blended clean + distorted
- Low-end foundation

VOCALS:
- Aggressive, rhythmic
- Synced with guitar stabs
- Dry mix

MIX:
- Very controlled low-end
- Clear separation
- Modern, precise
```

### Free VST Approach
```
Guitars: TSE 808 → Ignite Emissary → NadIR
Drums: Steven Slate Free, heavy processing
Bass: Clean DI + distorted blend
Vocals: Heavy compression, minimal reverb
```

## Tool

### Overall Production Style
- Dark, atmospheric
- Odd time signatures
- Space and dynamics
- Organic yet heavy

### Key Members
- Guitar: Adam Jones → See guitarists.md
- Bass: Justin Chancellor → See bassists.md
- Drums: Danny Carey → See drummers.md

### Production Approach
```
DRUMS:
- Natural room sound
- Blended acoustic + electronic
- Complex patterns
- Dynamic range preserved

GUITARS:
- Minimal effects
- Drop D or lower tunings
- Space between riffs
- Dynamic playing

BASS:
- Distorted with chorus
- Often leads melodically
- Prominent in mix

MIX:
- Dynamic (quiet to loud)
- Dark overall tone
- Room ambience important
- Stereo bass moments
```

### Free VST Approach
```
Guitars: Ignite Emissary (moderate gain)
Bass: Distortion + TAL-Chorus-LX
Drums: Natural samples, room reverb
Mix: Dynamic, not crushed
```

## Pink Floyd

### Overall Production Style
- Atmospheric, spacious
- Experimental sound design
- Long instrumental passages
- Emotional dynamics

### Key Members
- Guitar: David Gilmour → See guitarists.md
- Keyboards: Richard Wright → See keyboardists.md

### Production Approach
```
GUITARS:
- Long, modulated delays
- Smooth sustain
- Clean to light overdrive
- Expressive bends

KEYBOARDS:
- Atmospheric pads
- Hammond organ
- Minimoog leads
- Support, not dominate

DRUMS:
- Natural, dynamic
- Room sound
- Simple patterns, good feel

MIX:
- SPACE is crucial
- Long reverbs
- Stereo sound effects
- Dynamic range preserved
```

### Free VST Approach
```
Guitars: Clean amp + Valhalla Supermassive
Keys: Surge XT pads + Dexed organ
Effects: Valhalla Supermassive, TAL-Chorus
Mix: Lots of reverb, dynamic
```

## Metallica

### Overall Production Style
- Varies by album
- Heavy, aggressive
- Tight rhythm section
- Prominent bass (early) to scooped (later)

### Key Members
- Guitars: James Hetfield, Kirk Hammett
- Drums: Lars Ulrich → See drummers.md
- Bass: Cliff Burton / Robert Trujillo → See bassists.md

### Production Approach
```
BLACK ALBUM ERA:

GUITARS:
- Mesa Boogie Mark IIC+
- Tight rhythm, scooped mids
- Double/quad tracked
- Clear separation L/R

DRUMS:
- Natural sound + samples
- Punchy kick
- Room ambience

BASS:
- Supportive, follows guitar
- Clean tone
- (Note: Very quiet on later albums)

MIX:
- Punchy, powerful
- Controlled low-end
- Present vocals
```

### Free VST Approach
```
Guitars: Ignite Emissary, Mesa-style IR
Drums: Samples + room
Mix: Powerful, punchy
```

## Daft Punk

### Overall Production Style
- French house
- Vocoder vocals
- Analog synthesizers
- Disco/funk influences

### Production Approach
```
SYNTHS:
- Analog-style (or real analog)
- Filtered basslines
- Layered leads

DRUMS:
- Drum machine patterns
- Sidechain compression (pumping)
- 4/4 house beats

VOCALS:
- Vocoder (TAL-Vocoder - free)
- Robotic processing
- Talk box effects

BASS:
- Filtered analog bass
- Sidechain to kick

MIX:
- Warm, analog character
- Heavy sidechain compression
- Loud, polished
```

### Free VST Approach
```
Synths: TAL-NoiseMaker, Surge XT
Vocoder: TAL-Vocoder (free!)
Drums: Samples, heavy sidechain
Mix: Warm, pumping, loud
```

# Production Quick Reference

## By Genre

### Modern Metal (Meshuggah, Gojira, Periphery)
```
- Tight low-end
- TSE 808 before amp
- Heavy sample reinforcement
- Quad track guitars
- Modern, precise mix
```

### Classic Rock (Led Zeppelin, Queen)
```
- Room sound crucial
- Natural dynamics
- Layered guitars
- Vintage character
```

### Progressive (Tool, Dream Theater)
```
- Dynamic range
- Odd time signatures
- Complex arrangements
- Balance clarity with heaviness
```

### Electronic/Industrial (NIN, Daft Punk)
```
- Layered synths
- Heavy processing
- Sidechain compression
- Distortion as effect
```

### Alternative (Radiohead, Tame Impala)
```
- Experimental processing
- Blend electronic/organic
- Heavy effects
- Unconventional arrangements
```
