# Keyboardist Sounds - Free VST Recreation Guide

## Overview

Keyboard and synthesizer sounds are crucial for many genres. This guide covers famous keyboardists and how to recreate their signature sounds using free plugins.

---

## Jon Lord (Deep Purple)

### Signature Sound
- Hammond C3 through Marshall stack
- Overdriven organ tone
- Leslie speaker rotary effect
- Classical influences in rock

### Original Gear
- Hammond C3 organ
- Leslie 122 speaker
- Marshall guitar amps (for distortion)

### Free VST Chain
```
1. HaNon B70 (free Hammond emulation)
   OR: Dexed (can do organ-ish sounds)
2. Analog Obsession (overdrive)
3. Rotary speaker emulation:
   - TAL-Chorus-LX (fake Leslie)
   - Blue Cat's Chorus (modulate speed)
```

### REAPER FX Chain
```
ReaEQ:
- HPF at 80 Hz
- Boost 800 Hz - 1 kHz (bark)
- Boost 2-3 kHz (cut through guitars)

For overdrive:
- Softube Saturation Knob (heavy)
- Or Analog Obsession preamp
```

### Tips
- Play through guitar amp sim for dirt
- Leslie speed changes (slow/fast) for expression
- Key songs: "Highway Star", "Lazy"

## Rick Wakeman (Yes)

### Signature Sound

- Grand piano
- Multi-keyboard setup

### Free VST Recreation
```
Mellotron:
- Keyzone Classic (has Mellotron samples)
- VSCO2 Orchestra (free orchestral)

Moog lead:
- Surge XT (excellent free synth)
- TAL-NoiseMaker
- Dexed

Piano:
- Keyzone Classic (piano)
- Piano One (free Yamaha piano)
```

### Surge XT Moog-style Lead
```
Oscillator: Saw wave x2, slightly detuned
Filter: Moog-style ladder (24dB)
Filter cutoff: Medium-high, with resonance
Envelope: Fast attack, medium sustain
Add: Portamento (glide)
```

### Tips
- Layer multiple keyboards
- Dramatic swells with mod wheel
- Key songs: "Roundabout", "Close to the Edge"

## Vangelis

### Signature Sound
- Massive, cinematic pads
- CS-80 synthesizer
- Blade Runner soundscapes
- Emotional, expressive playing

### Free VST Recreation
```
1. Surge XT (can do CS-80 style)
2. Vital (lush pads)
3. Valhalla Supermassive (essential!)
4. Dexed (brass-like FM)

CS-80 style patch in Surge XT:
- Two oscillators (saw + pulse)
- Sub oscillator
- Filter with resonance
- Slow attack, long release
- Lots of modulation
```

### REAPER Processing
```
ReaEQ:
- Smooth frequency response
- Gentle high shelf boost (air)

Effects:
- Valhalla Supermassive (huge reverbs)
- TAL-Chorus-LX (width)
- Long delay tails
```

### Tips
- Play expressively with aftertouch/mod wheel
- Build slowly, cinematic arcs
- Key songs: "Blade Runner Main Theme", "Chariots of Fire"
