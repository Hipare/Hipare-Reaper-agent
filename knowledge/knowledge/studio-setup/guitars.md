## PICKUP CHARACTERISTICS

### Passive Single Coils
```
Output: 100-200mV
Sound: Bright, clear, articulate, 60Hz hum
Interface gain: HIGH (11-1 o'clock)
Best for: Clean, funk, blues, country
```

### Passive Vintage Humbuckers (PAF-style)
```
Output: 200-350mV
Sound: Warm, balanced, classic rock
Interface gain: MEDIUM (10-12 o'clock)
Best for: Blues, rock, jazz
Examples: Gibson PAF, Burstbucker, SD '59, DiMarzio PAF
```

### Passive High-Output Humbuckers
```
Output: 400-600mV
Sound: Aggressive, tight, metal-focused
Interface gain: MEDIUM-LOW (10-11 o'clock)
Best for: Metal, hard rock, djent
Examples: SD Distortion, JB, Nazgul; DiMarzio D-Activator; Cepheus
```

### Active Humbuckers (EMG, Fishman)
```
Output: 1-2V (VERY HIGH!)
Sound: Tight, compressed, quiet, modern metal
Interface gain: LOW (9 o'clock or less!)
Best for: Metal, djent, technical playing
Examples: EMG 81/85, EMG 57/66, Fishman Fluence Modern
Warning: Can easily clip interface input!
```

## AI DETECTION RULES

Kun käyttäjä mainitsee kitaramerkin/mallin:

1. **Etsi yllä olevasta listasta** → tunnista tyyppi
2. **Jos löytyy** → käytä sen tyypin asetuksia
3. **Jos EI löydy** → kysy tarkennusta tai arvioi nimen perusteella:
   - "Hellraiser", "EMG" → Active
   - "RG", "Jackson", "Schecter" → Superstrat/high-output
   - "Les Paul", "LP", "SG" → LP-type
   - "Strat", "Stratocaster" → Strat-type
   - Numero 7 tai 8 nimessä → Extended range

4. **Pickup-oletus jos ei kerrottu:**
   - Strat/Tele = passive single coil
   - Les Paul/SG = passive humbucker
   - Superstrat = high-output passive
   - 7-string = high-output passive TAI EMG
   - 8-string = high-output passive TAI EMG
   - "Hellraiser" = EMG (active)

5. **Kysy vain jos kriittistä:**
   - "Onko kitarassasi aktiivi- vai passiivi-mikit?"
   - (Vaikuttaa merkittävästi gain stagingiin)

## AI INSTRUCTIONS

Kun käyttäjä mainitsee kitaran äänittämisen:

1. **Tunnista merkki/malli** → etsi tietokannasta
2. **Määritä tyyppi** → SC, HB, high-output, active, extended
3. **Anna oikeat gain-asetukset** → tyypin mukaan
4. **Suosittele amp sim** → genreen ja tyyppiin sopiva
5. **Luo DI + monitor -setup** → puhdas DI + efektoitu monitori

Jos käyttäjä ei kerro pickupeja:
→ Käytä oletusta tyypin mukaan
→ Kysy vain jos epäselvä tai kriittinen (active vs passive)
