# Electronic Music Production Guide

## Overview

Electronic music is produced primarily with synthesizers, samplers, and digital tools. Focus on sound design, rhythm, and texture.

## Subgenres Covered

- **House** (Daft Punk, Disclosure, Calvin Harris)
- **Techno** (Kraftwerk, Richie Hawtin, Charlotte de Witte)
- **EDM/Big Room** (Avicii, Martin Garrix, Swedish House Mafia)
- **Synthwave** (Kavinsky, The Midnight, Perturbator)
- **Ambient** (Brian Eno, Boards of Canada, Tycho)
- **Drum & Bass** (Pendulum, Noisia, Netsky)
- **Dubstep** (Skrillex, Excision, Zeds Dead)
- **Trance** (Armin van Buuren, Above & Beyond)
- **Lo-Fi Hip Hop** (Nujabes, J Dilla influenced)

## Kick Drum Production

### House/Techno Kick

**Characteristics:**
- Punchy attack (2-5 kHz)
- Deep sub (40-60 Hz)
- Minimal mid frequencies

**Layering Technique:**
```
Layer 1: Sub layer (sine wave, 40-60 Hz)
Layer 2: Punch layer (short click, 100-150 Hz)
Layer 3: Attack layer (transient, 2-5 kHz)
```

**Processing:**
```
EQ: Cut 200-400 Hz, boost 50-60 Hz and 3-4 kHz
Compression: Fast attack, fast release, parallel processing
Saturation: Light for warmth
```

### EDM/Big Room Kick

**Characteristics:**
- Massive sub impact
- Sharp, clicky attack
- Tight, controlled

**Processing:**
```
Transient shaper: Increase attack
Compression: Heavy, fast
Saturation: More aggressive
Limiter: Control peaks
```

### Drum & Bass Kick

**Characteristics:**
- Shorter, punchier
- Less sub than house
- Tight decay

## Bass Production

### Sub Bass

**Key Principles:**
- Mono below 150 Hz
- Simple waveforms (sine, triangle)
- Sidechain to kick
- Clean, not distorted in sub region

**Sidechaining:**
```
1. Insert compressor on bass
2. Sidechain from kick
3. Ratio: 4:1 to inf:1
4. Fast attack, medium release
5. Adjust to create pump
```

### Dubstep/Growl Bass

**Design Approach:**
```
1. Start with saw/square oscillators
2. Heavy filter modulation (LFO, envelope)
3. FM synthesis for metallic textures
4. Distortion/saturation
5. Formant filtering for vowel sounds
```

**Key: Serum, Vital, or similar wavetable synth**

### Synthwave Bass

**Characteristics:**
- Fat analog-style
- Pulse width modulation
- Filter envelope for movement
- Often octave layers

**Reference:** Kavinsky, Perturbator

## Synth Design

### Lead Sounds

**Supersaw Lead:**
```
1. 7+ detuned saw oscillators
2. Unison with slight detune
3. Filter: Low pass, some resonance
4. Reverb and delay for space
5. Sidechain for pump
```

**Pluck Lead:**
```
1. Saw or square oscillator
2. Fast attack, short decay envelope
3. Filter envelope following amp
4. Short reverb
5. Chorus for width
```

### Pad Sounds

**Analog-Style Pad:**
```
1. 2-4 oscillators (saw, pulse)
2. Detune for thickness
3. Low pass filter with slow LFO
4. Long attack, long release
5. Chorus, reverb, delay
```

**Ambient Pad:**
```
1. Wavetable with movement
2. Grain/texture layer
3. Heavy reverb
4. Modulating filter
5. Stereo width processing
```

### Arp Patterns

**Classic Arp Setup:**
```
1. Simple waveform
2. Arpeggiator: 1/16 notes
3. Filter with envelope
4. Delay (tempo synced)
5. Sidechain for rhythm
```

## Drum Programming

### House Beat Foundation

```
Kick: 1, 1+, 2, 2+, 3, 3+, 4, 4+ (four on floor)
Clap/Snare: 2, 4
Hi-hat: Every 1/8 or 1/16
Open hat: Offbeats
```

### Techno Variations

```
Kick: Four on floor with variations
Percussion: Lots of layers, polyrhythms
Hi-hats: Complex patterns
Rides: For energy
```

### D&B Beat (170+ BPM)

```
Kick: Syncopated, not on every beat
Snare: 2 and 4 (half-time feel)
Hi-hats: Fast, complex
Ghost notes: Essential
```

### Trap/Hip-Hop Beat

```
Kick: 808 with long sub tail
Snare: 2 and 4
Hi-hats: Rolls, triplets
Tempo: 130-150 BPM (half-time feel)
```

## Arrangement

### EDM Drop Structure

```
Intro (16-32 bars) → Build (8-16 bars) → Drop (16-32 bars) → Breakdown (8-16 bars) → Build → Drop → Outro
```

**Build-up Techniques:**
- Rising filter sweeps
- Snare rolls increasing in frequency
- White noise risers
- Removing kick before drop
- Pitch risers
- Reverse cymbals/impacts

**Drop Techniques:**
- Full arrangement hits
- Sidechain pumping
- Heavy low end
- Main melody/hook

### House/Techno Structure

```
Intro (32 bars) → Verse (32) → Breakdown (16) → Build (8) → Peak (32) → Breakdown → Peak → Outro (32)
```

**More gradual, DJ-friendly structure**

### Ambient Structure

```
Free-form, evolving textures
Long sections (2-4 minutes)
Gradual changes
Focus on mood and atmosphere
```

## Mixing Electronic Music

### Frequency Balance

**Sub Bass (20-60 Hz):**
- Only kick and bass
- Mono
- Controlled, not overwhelming

**Bass (60-250 Hz):**
- Bass fundamental
- Kick punch
- Keep clean

**Mids (250 Hz - 5 kHz):**
- Synths, vocals, melody
- Most content lives here
- Careful EQ to prevent masking

**Highs (5-20 kHz):**
- Hi-hats, cymbals, air
- Presence and brightness

### Sidechain Compression

**Standard Setup:**
```
1. Insert compressor on synths/bass
2. Sidechain from kick
3. Ratio: 4:1 to inf:1
4. Attack: Fast (0.1-5 ms)
5. Release: Adjust to tempo (50-200 ms)
```

**Alternative: Volume automation**
- Draw volume envelope manually
- More control than compressor
- LFO tools in some DAWs

### Stereo Image

**Keep Centered:**
- Kick
- Bass
- Snare
- Lead vocal

**Spread Wide:**
- Pads
- Background synths
- Reverbs
- Delays
- Hi-hats (sometimes)

### Loudness

**Target LUFS by subgenre:**
| Genre | Target |
|-------|--------|
| House/Techno | -8 to -10 LUFS |
| EDM/Big Room | -6 to -8 LUFS |
| Ambient | -14 to -18 LUFS |
| D&B | -8 to -10 LUFS |
| Dubstep | -6 to -8 LUFS |

## Genre-Specific Tips

### Synthwave

**Key Elements:**
- Fat, analog-style bass
- Gated reverb drums (80s style)
- Arpeggiators
- Vintage synth sounds
- Drum machines (LinnDrum, 808, 909)

**Processing:**
- Chorus on everything
- Gated reverb on snare
- Tape saturation
- Analog-style mixing

### Ambient

**Key Elements:**
- Texture over rhythm
- Long reverbs
- Granular processing
- Field recordings
- Slow evolution

**Processing:**
- Heavy reverb (4s+)
- Delay feedback
- Modulation
- Gentle compression

### Lo-Fi Hip Hop

**Key Elements:**
- Dusty, vinyl texture
- Relaxed drums
- Jazz samples
- Tape saturation
- Imperfection is key

**Processing:**
- Bit reduction
- Vinyl crackle
- Tape wow/flutter
- Sidechain pumping
- High cut (lo-fi filtering)

## Essential Effects

### Filter Automation

Most important in electronic music:
```
- Low pass sweeps for builds
- High pass for breakdowns
- Resonance for emphasis
- Modulation for movement
```

### Reverb Usage

**Per Element:**
| Element | Reverb Type | Amount |
|---------|-------------|--------|
| Kick | None/minimal | Dry |
| Snare/Clap | Plate/room | Light-medium |
| Hi-hats | Room | Very light |
| Synths | Hall/plate | Medium-heavy |
| Pads | Hall/shimmer | Heavy |

### Delay Usage

**Tempo-synced common:**
- 1/8 note for energy
- Dotted 1/8 for bounce
- 1/4 note for epic
- Ping-pong for width

## Common Mistakes

1. **Too much sub**: Overwhelms mix, muddy
2. **No sidechain**: Static, no groove
3. **Over-processing**: Kills dynamics
4. **Ignoring arrangement**: All sound, no structure
5. **Mono bass not centered**: Phase issues
