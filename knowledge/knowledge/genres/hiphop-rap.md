# Hip-Hop & Rap Production Guide

## Overview

Hip-hop production centers on beats, bass, and vocals. From classic boom bap to modern trap, the focus is on rhythm, groove, and vocal delivery.

## Subgenres Covered

- **Boom Bap** (J Dilla, DJ Premier, Pete Rock)
- **Trap** (Metro Boomin, Mike Dean, 808 Mafia)
- **Lo-Fi Hip Hop** (Nujabes, Knxwledge)
- **West Coast** (Dr. Dre, DJ Quik)
- **Southern/Dirty South** (Three 6 Mafia, Mannie Fresh)
- **Cloud Rap** (Clams Casino, A$AP Rocky)
- **Drill** (UK/Chicago styles)

## Drum Production

### 808 Kick/Bass

**The Foundation of Modern Hip-Hop:**

**808 Processing:**
```
1. Start with clean 808 sample or synth
2. Tune to root note of track
3. Add saturation for upper harmonics
4. EQ: Boost 50-60 Hz, add presence at 150-200 Hz
5. Limit/clip for control
6. Sidechain to kick transient (if separate)
```

**Glide/Slide:**
- Use portamento for note slides
- Creates the classic trap sliding bass

**808 + Kick Layering:**
```
Option A: 808 with sub tail, separate kick transient
Option B: 808 that includes kick
Key: One dominates sub frequencies
```

### Boom Bap Drums

**Characteristics:**
- Punchy, swing feel
- Dusty, sampled aesthetic
- Prominent snare on 2 and 4
- Often from vinyl samples

**Kick:**
- Punchy, tight
- Some vinyl dust/saturation
- EQ: Less sub than trap, more punch

**Snare:**
- Fat, snappy
- Compression for punch
- Sometimes layered with clap

**Hi-Hats:**
- Steady 1/8 notes or variations
- Less rolls than trap
- Open hats for accents

### Trap Drums

**Characteristics:**
- 808 dominates
- Fast hi-hat rolls
- Snappy claps/snares on 2 and 4
- Triplet feel common

**Hi-Hat Programming:**
```
- 1/8, 1/16, 1/32 note variations
- Rolls and fills
- Velocity variation for human feel
- Open hat for accents
```

**Snare/Clap:**
- Layered snare + clap common
- Short, snappy
- On beats 2 and 4 (or 3 in half-time)

### Lo-Fi Hip Hop Drums

**Characteristics:**
- Dusty, warm
- Vinyl crackle and noise
- Relaxed swing
- Imperfect timing (intentional)

**Processing:**
```
1. Sample or program drums
2. Add vinyl noise layer
3. Bit crush lightly
4. Tape saturation
5. Roll off highs (lo-fi filter)
6. Add swing/groove
```

## Bass Production

### Trap 808 Bass

**Design:**
```
Oscillator: Sine or triangle
Attack: Fast
Decay: Long (800ms - 2s)
Sustain: 0 or low
Release: Medium-long
```

**Key Processing:**
- Tune to track root
- Saturation for harmonics
- EQ boost at 50-60 Hz
- Soft clip or limit

### Boom Bap Bass

**Options:**
1. **Sampled bass** from vinyl
2. **Synth bass** (Moog style)
3. **Live bass** recording

**Processing:**
```
Compression: 4:1, medium attack
EQ: Clean low end, some presence
Saturation: Analog warmth
```

### G-Funk Bass (West Coast)

**Characteristics:**
- Minimoog-style synth bass
- Melodic, playing lines
- Clean, warm tone
- Portamento for slides

## Sampling

### Chopping Samples

**Classic Technique:**
```
1. Find sample (soul, jazz, funk records)
2. Identify chops (drum hits, melodic phrases)
3. Slice and rearrange
4. Time-stretch or pitch-shift as needed
5. Add effects (filter, saturation)
```

**In Reaper:**
- Use razor editing for slicing
- Dynamic split for drum hits
- Stretch markers for time adjustments

### Sample Clearance

**Note:** If releasing commercially, samples need clearance. Consider:
- Royalty-free sample packs
- Original composition
- Heavy manipulation (replay)

## Vocal Production

### Rap Vocals

**Recording:**
```
Mic: Large diaphragm condenser or SM7B
Preamp: Clean with character
Room: Treated, minimal reflections
Takes: Multiple, comp the best
```

**Signal Chain:**
```
HPF (80-100 Hz) → Compression → EQ → De-esser → Saturation → Effects
```

**EQ:**
| Frequency | Action |
|-----------|--------|
| 80-120 Hz | HPF cut |
| 200-300 Hz | Cut if muddy |
| 2-4 kHz | Boost for presence |
| 5-8 kHz | De-ess or cut |
| 10-12 kHz | Air boost |

**Compression:**
```
Ratio: 4:1 to 6:1
Attack: 10-20 ms
Release: Auto or medium
GR: 4-8 dB
Consider serial compression
```

### Vocal Effects

**Delay:**
```
- 1/8 or 1/4 note
- Low feedback (1-3 repeats)
- Filter the delays
- Subtle blend
```

**Reverb:**
```
- Plate or room
- Short decay (0.8-1.5s)
- Pre-delay: 30-50 ms
- Keep vocal present
```

**Ad-Libs:**
- Pan to sides
- More effects than lead
- Shorter/quieter
- Doubles for emphasis

### Autotune/Pitch Correction

**Melodic Rap (Drake, Future style):**
```
- Fast retune speed (0-15 ms)
- Allows natural vibrato at end of phrases
- Key/scale lock to track
```

**Subtle Correction:**
```
- Slower retune (30-100 ms)
- Natural sounding
- Preserves performance
```

## Arrangement

### Typical Structure

```
Intro (4-8 bars)
Verse 1 (16 bars)
Hook/Chorus (8 bars)
Verse 2 (16 bars)
Hook (8 bars)
Bridge/Break (8 bars)
Verse 3 (optional)
Hook/Outro (8+ bars)
```

### Keeping Interest

**Variation Techniques:**
- Add/remove elements each 4-8 bars
- Filter sweeps
- New percussion in verse 2
- Drop out for impact
- Effects throws

## Mixing Hip-Hop

### Balance

**Typical Priority:**
1. Kick/808 - Foundation
2. Vocals - Star of the show
3. Snare/Clap - Groove
4. Hi-hats - Rhythm
5. Melodics - Support

### Vocals Over Beat

**Creating Space:**
```
1. EQ pocket in beat for vocals (2-4 kHz dip)
2. Sidechain compression (subtle)
3. Automate beat volume during vocals
4. High-pass bass instruments during verses
```

### Low End Management

**Critical in Hip-Hop:**
```
- 808/kick and bass must not compete
- Use sidechain or arrangement
- Mono below 100-150 Hz
- High-pass everything else
```

### Loudness

**Target Levels:**
| Style | Target LUFS |
|-------|-------------|
| Boom Bap | -10 to -12 |
| Trap | -6 to -8 |
| Lo-Fi | -12 to -14 |
| Commercial | -8 to -10 |

## Genre-Specific Tips

### Trap

- **808**: Long, tuned, dominant
- **Hi-hats**: Complex rolls, velocity variation
- **Clap/Snare**: Layered, punchy
- **Arrangement**: Build-ups, drops
- **Reference**: Metro Boomin, Southside

### Boom Bap

- **Samples**: Soul, jazz chops
- **Drums**: Punchy, swing
- **Feel**: Head-nodding groove
- **Production**: Often stripped back
- **Reference**: J Dilla, Pete Rock

### Lo-Fi Hip Hop

- **Aesthetic**: Imperfect, dusty
- **Drums**: Relaxed, vinyl texture
- **Keys**: Jazz piano, Rhodes
- **Processing**: Tape saturation, vinyl noise
- **Reference**: Nujabes, Lofi Girl compilations

### Drill (UK/Chicago)

- **Tempo**: 140+ BPM (UK), 60-70 BPM (Chicago half-time)
- **808**: Slides and glides
- **Hi-hats**: Rapid patterns
- **Percussion**: Metallic stabs
- **Dark melodics**: Minor keys

## Common Mistakes

1. **Muddy low end**: 808 and bass fighting
2. **Weak vocals**: Not cutting through
3. **Over-compressed**: Losing dynamics
4. **No variation**: Same loop throughout
5. **Ignoring groove**: Drums too mechanical
