# Pop Production Guide

## Overview

Pop (popular music) is commercially-oriented music aimed at mass appeal. Characterized by catchy melodies, clear structures, and polished production.

## Subgenres Covered

- **Mainstream Pop** (Taylor Swift, Dua Lipa, The Weeknd)
- **Dance Pop** (ABBA, Madonna, Lady Gaga)
- **Synth Pop** (Depeche Mode, Pet Shop Boys, The 1975)
- **Acoustic Pop** (Ed Sheeran, John Mayer)
- **Power Pop** (Cheap Trick, Weezer)
- **Indie Pop** (MGMT, Vampire Weekend)
- **K-Pop** (BTS, BLACKPINK)

## Tempo and Structure

### Typical Tempos
- **Ballad:** 60-80 BPM
- **Mid-tempo:** 90-110 BPM
- **Uptempo:** 110-130 BPM
- **Dance Pop:** 120-130 BPM

### Song Structure
```
Intro (4-8 bars) → Verse (8-16 bars) → Pre-Chorus (4-8 bars) → Chorus (8-16 bars) → Verse → Pre-Chorus → Chorus → Bridge (8 bars) → Chorus → Outro
```

**Key Rule:** Hook/chorus within 30 seconds

### Section Guidelines
| Section | Bars | Purpose |
|---------|------|---------|
| Intro | 4-8 | Set mood, grab attention |
| Verse | 8-16 | Story, build |
| Pre-Chorus | 4-8 | Tension, lift |
| Chorus | 8-16 | Payoff, hook |
| Bridge | 8 | Contrast, new element |

## Drum Production

### Pop Drum Sound

**Characteristics:**
- Tight, punchy
- Often sample-augmented
- Clean, polished
- Consistent dynamics

**Kick:**
```
EQ:
- HPF: 30-40 Hz
- Boost: 60-80 Hz (sub)
- Cut: 300-400 Hz (boxiness)
- Boost: 3-5 kHz (click)

Compression: 4:1, medium attack
```

**Snare:**
```
EQ:
- HPF: 80-100 Hz
- Boost: 150-200 Hz (body)
- Cut: 400-600 Hz (boxiness)
- Boost: 3-5 kHz (snap)
- Boost: 8-10 kHz (air)

Compression: 4:1, fast attack for control
```

**Hi-Hats:**
```
- HPF: 300-400 Hz
- Boost: 8-12 kHz (shimmer)
- Keep consistent, programmed feel
```

### Programmed vs Live Drums

**Modern Pop:** Often hybrid
- Kick/snare samples for consistency
- Live hi-hats and cymbals for feel
- Layered for impact

## Bass Production

### Pop Bass Approach

**Characteristics:**
- Supportive, not dominant
- Locked with kick
- Clean, defined
- Simple patterns

**EQ:**
```
- HPF: 30-40 Hz
- Cut: 200-400 Hz (clean up)
- Boost: 700-1000 Hz (presence, if needed)
```

**Compression:**
```
Ratio: 4:1
Attack: 20-40 ms
Release: 100-200 ms
GR: 4-6 dB
```

### Synth Bass vs Electric

**Synth Bass:** Modern pop, dance pop
- Clean sine/saw waves
- Tight envelope
- Sidechain to kick

**Electric Bass:** Acoustic pop, rock-influenced
- DI + amp blend
- More natural feel

## Vocal Production

### Lead Vocal Chain

**Signal Chain:**
```
Mic → Preamp → HPF (80-100 Hz) → Compression → EQ → De-esser → Saturation → Effects
```

**EQ Guidelines:**
| Frequency | Action | Purpose |
|-----------|--------|---------|
| 80-100 Hz | HPF | Remove rumble |
| 200-400 Hz | Cut 2-4 dB | Reduce mud |
| 3-5 kHz | Boost 2-4 dB | Presence, clarity |
| 10-12 kHz | Boost 1-2 dB | Air, sparkle |

**Compression:**
```
Stage 1 (Leveling):
- Ratio: 3:1 - 4:1
- Attack: 10-30 ms
- Release: 100-200 ms
- GR: 3-6 dB

Stage 2 (Parallel, optional):
- Heavy compression blended in
```

### Vocal Effects

**Reverb:**
```
- Type: Plate or hall
- Decay: 1.2-2s
- Pre-delay: 30-50 ms
- Mix: 15-25%
```

**Delay:**
```
- Type: Stereo, tempo-synced
- Time: 1/4 or 1/8 note
- Feedback: 20-40%
- Filter: High cut on repeats
```

### Vocal Layering

**Doubles:**
- Pan L30/R30 or L50/R50
- Slightly lower in mix
- Tighter compression

**Harmonies:**
- 3rds and 5ths common
- Panned wide
- More reverb than lead

**Ad-libs:**
- Panned to sides
- More effects
- Lower in mix

### Pitch Correction

**Subtle (natural):**
```
- Retune speed: 50-100 ms
- Humanize: On
- Natural vibrato preserved
```

**Heavy (modern pop/effect):**
```
- Retune speed: 0-20 ms
- Creates characteristic "tuned" sound
- T-Pain, modern pop style
```

## Synth and Keys

### Common Sounds

**Pads:**
- Fill space
- Wide stereo
- Sidechain for pump
- Long attack, long release

**Plucks:**
- Rhythmic element
- Tempo-synced patterns
- Short decay

**Piano:**
- Acoustic for ballads
- Electric for modern pop
- Often layered with synths

### Processing

```
EQ: Cut lows below 100-150 Hz
Compression: Light, 2:1 for control
Reverb: Medium plate/hall
Sidechain: Common for pump effect
```

## Mixing Pop

### Balance Priority

1. **Vocals** - Always front and center
2. **Kick/Snare** - Groove foundation
3. **Bass** - Support, not dominate
4. **Hook elements** - Memorable parts
5. **Everything else** - Support role

### Frequency Balance

**Key Philosophy:** Make space for vocals

```
Sub (20-60 Hz): Kick and bass only
Low (60-200 Hz): Bass, kick body
Low-mid (200-500 Hz): Cut most things (vocal clarity)
Mid (500-2000 Hz): Body of instruments
Upper-mid (2-5 kHz): Vocal presence (prioritize!)
High (5-12 kHz): Air, sparkle
```

### Width Strategy

**Center (Mono):**
- Lead vocal
- Kick
- Snare
- Bass
- Main hook elements

**Wide (Stereo):**
- Doubles and harmonies
- Pads and synths
- Guitars (rhythm)
- Effects returns

### Bus Processing

**Drum Bus:**
```
- Glue compression: 2:1, 2-3 dB GR
- EQ: Subtle shaping
- Parallel compression optional
```

**Vocal Bus:**
```
- Light compression for glue
- EQ: Final shaping
- Saturation for warmth
```

**Master Bus:**
```
- EQ: Subtle high shelf (+1 dB @ 10 kHz)
- Compression: 2:1, 1-2 dB GR
- Limiter: Ceiling at -1 dBTP
```

### Target Loudness

| Style | Target LUFS |
|-------|-------------|
| Ballad | -10 to -12 |
| Mid-tempo | -8 to -10 |
| Dance Pop | -6 to -8 |
| Streaming optimized | -14 |

## Genre-Specific Tips

### Modern Mainstream (Taylor Swift, Dua Lipa)

- **Vocals**: Very present, multiple layers
- **Production**: Polished, every element intentional
- **Hooks**: Memorable, within 30 seconds
- **Dynamics**: Controlled but not flat
- **Reference**: Current Billboard hits

### Synth Pop (The 1975, Depeche Mode)

- **Synths**: Prominent, character-defining
- **Drums**: Often electronic/programmed
- **Bass**: Synth bass common
- **Atmosphere**: Important
- **Retro elements**: 80s influences

### Acoustic Pop (Ed Sheeran)

- **Guitar**: Acoustic, prominent
- **Vocals**: Natural, less processed
- **Production**: Intimate, organic
- **Dynamics**: More natural
- **Layers**: Build through song

### Dance Pop (Daft Punk, Calvin Harris)

- **Four on floor**: Kick on every beat
- **Sidechain**: Essential for pump
- **Builds/Drops**: Clear structure
- **Energy**: Consistent high energy
- **Bass**: Prominent, synth-driven

## Arrangement Tips

### Maintaining Interest

**Add/remove elements every 4-8 bars:**
```
Verse 1: Basic arrangement
Pre-Chorus: Add elements (synth, percussion)
Chorus: Full arrangement
Verse 2: Slightly bigger than V1
Chorus 2: Add harmony/double
Bridge: Strip down or contrast
Final Chorus: Biggest, all elements
```

### Common Arrangement Tools

- Filter automation (builds)
- Percussion additions
- Vocal ad-libs
- Counter-melodies
- Harmony additions
- Drum fills

## Effects Usage

### Reverb

| Element | Type | Decay | Amount |
|---------|------|-------|--------|
| Lead Vocal | Plate | 1.2-2s | Medium |
| Backing Vocals | Hall | 1.5-2.5s | More |
| Snare | Plate/Room | 0.8-1.5s | Light-Medium |
| Synths | Hall | 1.5-2.5s | Varies |

### Delay

| Element | Type | Time | Feedback |
|---------|------|------|----------|
| Lead Vocal | Stereo | 1/4 note | 20-30% |
| Guitar | Dotted 1/8 | Tempo-sync | 30-40% |
| Synth | Ping-pong | 1/8 note | 20-40% |

## Common Mistakes

1. **Buried vocals**: Vocals must be front and center
2. **No hook**: Chorus not memorable enough
3. **Over-compression**: Kills dynamics and energy
4. **Muddy low-mids**: 200-400 Hz buildup
5. **Too much reverb**: Vocals pushed back
6. **No contrast**: Verse and chorus sound same
7. **Late hook**: Takes too long to reach chorus
