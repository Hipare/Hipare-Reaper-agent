# Rock Production Guide

## Overview

Rock encompasses a vast range of styles from clean jangly tones to heavy distortion. Key elements: guitar-driven sound, solid rhythm section, and dynamic arrangements.

## Subgenres Covered

- **Classic Rock** (Led Zeppelin, Pink Floyd, The Who)
- **Alternative Rock** (Radiohead, Foo Fighters, Arctic Monkeys)
- **Indie Rock** (The Strokes, Arctic Monkeys, Tame Impala)
- **Grunge** (Nirvana, Pearl Jam, Alice in Chains)
- **Punk Rock** (Ramones, Green Day, Blink-182)
- **Post-Rock** (Sigur Rós, Explosions in the Sky)
- **Hard Rock** (AC/DC, Guns N' Roses, Van Halen)

## Guitar Production

### Classic Rock Tone

**Signal Chain:**
```
Guitar → Tube Amp (edge of breakup) → Mic'd Cabinet → EQ → Light Compression
```

**Key Characteristics:**
- Natural tube amp saturation
- Dynamic response to playing
- Less compressed than modern tones
- Room ambience important

**Amp Settings (Marshall-style):**
```
Gain: 4-6/10
Bass: 5-6
Mid: 6-7
Treble: 6-7
Presence: 5-6
```

### Alternative/Indie Tone

**Signal Chain:**
```
Guitar → Pedals (modulation, reverb) → Clean/Edge Amp → EQ → Compression
```

**Key Characteristics:**
- Cleaner base tones
- Lots of effects (chorus, delay, reverb)
- Dynamic contrast between sections
- Often multiple guitar textures

**Amp Settings (Fender/Vox style):**
```
Gain: 3-5/10
Bass: 5
Mid: 5-6
Treble: 6
Reverb: As needed
```

### Grunge Tone

**Signal Chain:**
```
Guitar → Distortion Pedal → Amp → Heavy Room Mics → EQ → Compression
```

**Key Characteristics:**
- Heavy but muddy (intentional)
- Loud/quiet dynamics
- Raw, unpolished feel
- Room sound important

**Reference:** Nirvana "Nevermind", Alice in Chains "Dirt"

### Punk Rock Tone

**Signal Chain:**
```
Guitar → High Gain → Fast, Direct Recording → Minimal Processing
```

**Key Characteristics:**
- Fast, aggressive playing
- Less polished than other rock
- Energy over perfection
- Minimal overdubs

## Bass Production

### Standard Rock Bass

**EQ Guidelines:**
| Frequency | Action |
|-----------|--------|
| 40-80 Hz | Sub foundation |
| 80-150 Hz | Main body |
| 200-400 Hz | Cut mud |
| 700-1500 Hz | Boost for presence |
| 2-4 kHz | String definition |

**Compression:**
```
Ratio: 4:1
Attack: 20-40 ms
Release: 100-200 ms
GR: 4-6 dB
```

### Pick vs Fingers

- **Pick**: More attack, brighter, cuts through
- **Fingers**: Warmer, rounder, more fundamental
- Genre typically determines choice

## Drum Production

### Rock Drum Sound

**Kick:**
- Less clicky than metal
- More boom, natural resonance
- HPF: 40-50 Hz
- Boost: 60-100 Hz (thump)
- Cut: 300-400 Hz (boxiness)
- Light boost: 2-4 kHz (attack)

**Snare:**
- Ringy, open sound acceptable
- Fat, punchy character
- HPF: 80 Hz
- Boost: 150-200 Hz (body)
- Boost: 2-4 kHz (crack)
- Possible 5-8 kHz boost (wires)

**Toms:**
- Full, resonant
- Less gated than metal
- Natural decay

**Overheads:**
- Important for overall kit sound
- HPF: 200-400 Hz
- Capture cymbals and room

**Room Mics:**
- Essential for rock
- Compress heavily for size
- Blend to taste

### Room Sound

Room mics are crucial in rock:
```
1. Set up room mics 5-15 feet from kit
2. Compress heavily (10:1+, fast attack)
3. Blend with close mics for size
4. Adds depth and excitement
```

## Vocals

### Rock Lead Vocal

**Signal Chain:**
```
Mic (SM7B, U87) → Preamp → HPF → Compression → EQ → Saturation → Reverb/Delay
```

**EQ:**
- HPF: 80-120 Hz
- Cut: 200-400 Hz (if muddy)
- Boost: 2-4 kHz (presence)
- Boost: 10-12 kHz (air)

**Compression:**
```
Ratio: 4:1 to 6:1
Attack: 10-30 ms
Release: Auto or 100-200 ms
GR: 4-8 dB
```

### Background Vocals

- Wide panning (L/R)
- HPF higher (150-200 Hz)
- More compression
- Reverb to push back
- Often doubled or stacked

## Mixing Approach

### Classic Rock Mix

```
Drums: Natural, roomy, dynamic
Bass: Present but not overpowering
Guitars: Wide stereo, dynamic
Vocals: Up front but integrated
Overall: Dynamic, breathing
```

**Target LUFS:** -12 to -14

### Modern Rock Mix

```
Drums: Tighter, more controlled
Bass: Present, defined
Guitars: Thick, layered
Vocals: Very present, processed
Overall: Louder, less dynamic
```

**Target LUFS:** -10 to -12

## Genre-Specific Tips

### Alternative Rock (Radiohead, Foo Fighters)

- **Dynamics**: Crucial for impact
- **Textures**: Layered guitars, synths
- **Production**: Polished but organic
- **Drums**: Often natural, sometimes electronic elements

### Indie Rock (The Strokes, Arctic Monkeys)

- **Guitars**: Jangly cleans, crunchy drives
- **Drums**: Tight, punchy, often minimal
- **Bass**: Present, melodic
- **Overall**: Lo-fi aesthetic sometimes desired

### Post-Rock (Explosions in the Sky)

- **Build-ups**: Everything about dynamics
- **Reverb**: Lots of it
- **Delay**: Crucial for atmosphere
- **Guitars**: Clean swells, ambient textures
- **Drums**: Subtle to explosive

### Grunge (Nirvana, Pearl Jam)

- **Raw energy**: Over polish
- **Dynamics**: Quiet verse, loud chorus
- **Guitar tones**: Heavy, sludgy
- **Mix**: Intentionally rough

### Hard Rock (AC/DC, GNR)

- **Guitars**: Crunchy, rhythmic
- **Drums**: Big, powerful
- **Bass**: Locked with kick
- **Vocals**: Aggressive, present

## Effects Usage

### Guitar Effects

**Delay:**
- Tempo-synced (1/8, dotted 1/8)
- Moderate feedback (30-50%)
- High cut for vintage feel

**Reverb:**
- Plate or room
- Medium decay (1-2s)
- Pre-delay to maintain clarity

**Modulation:**
- Chorus for jangle
- Phaser for movement
- Tremolo for atmosphere

### Vocal Effects

**Reverb:**
- Plate most common
- 1-2s decay
- Pre-delay: 30-60 ms

**Delay:**
- 1/8 note common
- Filtered repeats
- Blend subtly

**Doubling:**
- Natural doubles or ADT
- Wide panning
- Choruses especially

## Common Mistakes

1. **Over-compressing**: Rock needs dynamics
2. **Too much low end**: Mud city
3. **Neglecting room sound**: Loses energy
4. **Over-processing**: Kills organic feel
5. **Wrong reverb**: Too long = washy

## Reaper Workflow

1. Group drums, guitars, bass, vocals in folders
2. Use room mic bus with heavy compression
3. Parallel compression on drums
4. Automate dynamics (quiet verses, loud choruses)
5. Reference constantly
