# Orchestral/Classical Production Guide

## Overview

Orchestral music encompasses symphony orchestra, chamber ensembles, and film/game scoring. Focus on dynamic range, natural acoustics, spatial depth, and the authentic sound of acoustic instruments.

## Subgenres Covered

- **Classical** (Mozart, Beethoven, Bach)
- **Romantic** (Tchaikovsky, Mahler, Wagner)
- **Modern Classical** (Stravinsky, Bartók, Glass)
- **Film Score** (John Williams, Hans Zimmer, Howard Shore)
- **Game Music** (Nobuo Uematsu, Jeremy Soule)
- **Trailer Music** (Epic, hybrid orchestral)
- **Neo-Classical** (Ólafur Arnalds, Nils Frahm)

## Orchestra Composition

### Full Symphony Orchestra

**Strings (typically 60+ players):**
```
1st Violins: 16
2nd Violins: 14
Violas: 12

```

**Woodwinds (in pairs typically):**
```
Flutes: 2-3 (+ piccolo)

```

**Brass:**
```
French Horns: 4-6

```

**Percussion:**
```

Triangle, tambourine, etc.
```

**Other:**
```

Piano/Celesta: As needed

```

## Recording Philosophy

### Key Principles

1. **Capture the performance** - Recording, not creating
2. **Room is essential** - Concert hall acoustics
3. **Preserve dynamics** - Full orchestral range (60+ dB)
4. **Minimal processing** - Trust the source
5. **Stereo/Surround** - Spatial presentation critical

### Recording Formats
```
Stereo: Standard release format
5.1 Surround: Film, Blu-ray
7.1 or higher: Immersive cinema
Binaural: Headphone listening
```

## Microphone Techniques

### Main Pair Systems

**Decca Tree:**
```
- 3 omnidirectional mics
- Center + L/R spread (2m apart)
- Classic orchestral sound
- Rich, enveloping
```

**ORTF:**
```
- 2 cardioid mics
- 110° angle, 17cm apart
- Good stereo image
- Natural perspective
```

**AB Spaced Pair:**
```
- 2 omnidirectional mics
- 2-3m apart
- Wide, spacious sound
- Good for large ensembles
```

### Spot Microphones

**Purpose:** Add definition to distant sections

```
Strings: Overhead or per section
Woodwinds: Individual or pairs
Brass: Careful - naturally loud
Percussion: Spot for timpani, individual instruments
Harp: Close mic for definition
```

**Blending:** Spot mics delayed to align with main pair

## EQ Guidelines

### Philosophy

**Minimal intervention.** Orchestral EQ is about fixing problems, not creating sounds.

### Strings
```
HPF: 60-80 Hz (basses lower)
Rumble: Cut below 40 Hz
Harshness: Cut 2-4 kHz if excessive
Air: Gentle boost 10-12 kHz (violins "silk")
General: Leave alone if recorded well
```

### Woodwinds
```
HPF: 100-150 Hz (instrument dependent)
Oboe: Cut 800-1000 Hz if too nasal
Flute: Boost 8-12 kHz for air (careful)
Clarinet: Generally leave alone
```

### Brass
```
HPF: 60-100 Hz
Harshness: Cut 2-4 kHz if strident
Horns: Boost 200-400 Hz for warmth (careful)
Generally: Brass is naturally bright
```

### Timpani
```
HPF: 30-40 Hz only
Fundamental: 80-150 Hz - don't cut!
Attack: 2-4 kHz
Roll-off: No high boost needed
```

## Compression Guidelines

### Philosophy

**Dynamics ARE the music.** Orchestral music ranges from pianissimo to fortissimo - potentially 60+ dB dynamic range. Compression destroys this.

### When Compression is Used
```
Situation: Broadcast, background music
Ratio: 1.5:1 - 2:1 (very gentle)
Attack: Very slow (50+ ms)
Release: Very slow (300-500+ ms)
GR: 1-2 dB maximum
```

### Mastering
```
Limiter: Safety only, not for loudness
True Peak: -1 dBTP ceiling
Dynamic Range: DR 14-20 is normal
DO NOT COMPETE with pop loudness
```

### Target Loudness

| Format | Target |
|--------|--------|
| Album Release | -18 to -24 LUFS |
| Film Score (theatrical) | Per dialogue norm |
| Streaming | Let normalization handle it |
| Background/Library | -14 to -16 LUFS |

## Reverb and Space

### Philosophy

**The hall IS the reverb.** Live orchestral recordings in concert halls often need no added reverb.

### For Studio/Sample Productions

**Reverb Type:** Convolution (impulse response)
```
Famous Halls IRs:
- Vienna Musikverein
- Boston Symphony Hall
- Concertgebouw Amsterdam
- Abbey Road Studio 1
```

**Settings:**
```
Decay: 1.8-3s (matches real halls)
Pre-delay: 20-40 ms (for clarity)
Goal: Place all instruments in same space
```

### Creating Depth (Front to Back)
```
Front: Strings (less reverb, more direct)
Middle: Woodwinds
Back: Brass, percussion (more reverb)
Use send amounts to create depth
```

### Delay
```
Generally: NOT USED in classical
Exception: Some experimental/modern works
```

## Mixing Orchestral

### Balance Goals

1. **Natural** - Sounds like real orchestra in real hall
2. **Dynamic** - Full range preserved
3. **Depth** - Front to back staging
4. **Width** - Appropriate stereo spread
5. **Clarity** - Each section audible

### Stereo Image

**Traditional layout:**
```
Hard Left: 1st Violins
Left: 2nd Violins, Violas (partial)
Center: Woodwinds (slightly spread)
Right: Cellos, Violas (partial)
Right: Basses
Wide: Brass, Percussion
```

### Depth Mixing (for sample productions)

```
Close → Far achieved by:
1. Reverb send amount (more = further)
2. High frequency roll-off (air absorption)
3. Volume (quieter = further, subtle)
4. Delay to main reverb (distance delay)
```

### Humanization Tips

```
1. Vary velocity (no machine-gun effect)
2. Slight timing variations
3. Use multiple articulations
4. Avoid quantized rhythms
5. Use expression/dynamics CC
6. Layer different mic positions
```

### Articulations to Know
```
Legato: Smooth, connected
Staccato: Short, detached
Spiccato: Bounced bow (strings)
Pizzicato: Plucked strings
Tremolo: Rapid bow/notes
Marcato: Accented
Con sordino: Muted
```

## Subgenre-Specific Tips

### Film Scoring (John Williams style)

- **Orchestration**: Full, traditional
- **Dynamics**: Important for emotion
- **Themes**: Memorable, leitmotifs
- **Balance**: With dialogue considered

### Modern Film (Hans Zimmer style)

- **Hybrid**: Orchestra + synths + samples
- **Low end**: Emphasis on sub bass
- **Processing**: More acceptable
- **Sound design**: Integrated

### Trailer Music

- **Impact**: Huge, epic
- **Processing**: Heavy compression acceptable
- **Layers**: Orchestra + synth + percussion
- **Build**: Escalating intensity
- **Volume**: Loud (-10 to -8 LUFS)

### Neo-Classical (Ólafur Arnalds)

- **Intimate**: Small ensembles
- **Electronics**: Integrated tastefully
- **Space**: Room ambience important
- **Dynamics**: Still preserved

## Common Mistakes

1. **Over-compressing**: Classical needs full dynamics
2. **Wrong reverb**: Artificial, inappropriate decay
3. **Too loud**: Classical isn't loud
4. **Ignoring depth**: Flat front-to-back
5. **Machine-gun samples**: Unvaried velocities
6. **Bad balance**: Following sample defaults
7. **Phase issues**: Spot mics not aligned

## Production Tips

1. **Study scores**: Understand orchestration
2. **Reference constantly**: Know the real sound
3. **Room first**: Acoustics matter most
4. **Less processing**: Trust the performance
5. **Dynamics sacred**: Never squash them
6. **Learn instruments**: Know their ranges and colors
7. **Mono compatibility**: Check phase relationships
