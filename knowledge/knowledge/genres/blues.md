# Blues Production Guide

## Overview

Blues is an African-American musical genre rooted in emotion, storytelling, and the "blue notes." Characterized by the 12-bar structure, shuffle rhythm, and expressive playing. Production emphasizes warmth, dynamics, and authentic feel.

## Subgenres Covered

- **Delta Blues** (Robert Johnson, Son House)
- **Chicago Blues** (Muddy Waters, Howlin' Wolf)
- **Texas Blues** (Stevie Ray Vaughan, Albert Collins)
- **British Blues** (Eric Clapton, Fleetwood Mac)
- **Jump Blues** (Louis Jordan, T-Bone Walker)
- **Blues Rock** (Joe Bonamassa, Gary Clark Jr.)
- **Modern Blues** (Christone "Kingfish" Ingram)

## Tempo and Structure

### Typical Tempos
- **Slow Blues:** 50-70 BPM
- **Medium Shuffle:** 90-120 BPM
- **Up-tempo:** 120-150 BPM

### 12-Bar Blues Structure
```
| I7  | I7  | I7  | I7  |    (bars 1-4)
| IV7 | IV7 | I7  | I7  |    (bars 5-8)
| V7  | IV7 | I7  | V7  |    (bars 9-12, turnaround)
```

**In Key of A:**
```
| A7  | A7  | A7  | A7  |
| D7  | D7  | A7  | A7  |
| E7  | D7  | A7  | E7  |
```

### Variations
- **Quick Change:** IV chord in bar 2
- **8-Bar Blues:** Compressed form
- **16-Bar Blues:** Extended form
- **Minor Blues:** i - iv - V

### Shuffle Rhythm

**The Blues Feel:**
```
Straight 8ths: | 1 & 2 & 3 & 4 & |
Shuffle:       | 1 . a 2 . a 3 . a 4 . a |
(triplet feel, middle note silent)
```

**Key:** "Behind the beat" - relaxed, not rushed

## Harmony and Scales

### Blue Notes
```
Standard Major: 1 - 2 - 3 - 4 - 5 - 6 - 7
Blues adds: b3, b5, b7 (microtonal bends)
```

### Blues Scale
```
A Blues: A - C - D - Eb - E - G - A
        (1 - b3 - 4 - b5 - 5 - b7 - 1)
```

### Chord Types
- **Dominant 7ths:** All chords are typically 7ths
- **9th chords:** Added color (especially I9)
- **Extended:** 13ths for jazz-blues

## Instrumentation

### Classic Blues Band
- **Electric Guitar:** Lead and rhythm
- **Bass:** Electric or upright
- **Drums:** Shuffle groove, brushes or sticks
- **Vocals:** Emotional, storytelling
- **Harmonica:** Classic blues timbre
- **Piano/Organ:** Boogie patterns, Hammond B3

### Frequency Ranges
| Instrument | Range | Character |
|------------|-------|-----------|
| Electric Guitar | 80 Hz - 6 kHz | Warmth + grit |
| Bass | 40-400 Hz | Walking, supportive |
| Vocals | 100 Hz - 8 kHz | Raw, not polished |
| Hammond B3 | 100 Hz - 5 kHz | Leslie rotation |
| Harmonica (amped) | 200 Hz - 4 kHz | Gritty, overdriven |

## Guitar Production

### Classic Blues Tone

**Signal Chain:**
```
Guitar → Tube Screamer/OD → Tube Amp (edge of breakup) → Mic'd Cabinet → EQ
```

**Key Characteristics:**
- Warm, not harsh
- Dynamic response to playing
- Slight breakup, not heavy distortion
- Natural tube saturation

### Amp Styles by Subgenre

**Chicago Blues (Muddy Waters):**
```
Amp: Fender Bassman, Twin
Gain: Medium breakup
Character: Fat, warm, punchy
```

**Texas Blues (SRV):**
```
Amp: Fender Vibroverb, Dumble
Pedals: Tube Screamer for boost
Character: Aggressive, dynamic, loud
```

**British Blues (Clapton/Bluesbreakers):**
```
Amp: Marshall Bluesbreaker, JTM45
Character: Crunchy, mid-focused
"Woman tone": Roll off tone knob
```

### Guitar EQ
```
HPF: 80-100 Hz (remove mud)
Boost: 100-200 Hz (warmth)
Cut: 300-500 Hz (boxiness, if needed)
Presence: 2-4 kHz (bite, careful not too much)
DO NOT over-brighten: Blues is warm, not shimmery
```

### Compression
```
Ratio: 3:1 - 4:1
Attack: Slow (20-40 ms) - preserve pick attack
Release: 150-250 ms
GR: 3-5 dB
Goal: Even out, not squash
```

## Vocal Production

### Blues Vocal Character

**Key Traits:**
- Emotional, raw
- Dynamic
- Imperfect (intentionally)
- Storytelling

### Signal Chain
```
Mic → Preamp (with character) → HPF → Compression → EQ → Saturation → Effects
```

### EQ
```
HPF: 80-100 Hz
Warmth: Boost 150-250 Hz (optional)
Cut: 400-600 Hz (if boxy)
Presence: 2-4 kHz (only if needed)
Keep raw: Don't over-polish
```

### Compression
```
Ratio: 3:1 - 4:1
Attack: 15-30 ms
Release: 100-200 ms
GR: 4-6 dB
Note: Blues vocals should be dynamic!
```

### Effects

**Slapback Delay (Classic Blues):**
```
Time: 80-150 ms
Feedback: 0-20% (1-2 repeats)
Use: Guitar, vocals
Character: Adds depth, vintage feel
```

**Reverb:**
```
Type: Spring reverb (classic) or plate
Decay: 1-2 seconds
Mix: 15-25%
Don't drown the vocal
```

## Bass Production

### Approach

**Walking Bass:** Melodic, moving line
**Support:** Lock with kick drum
**Character:** Round, warm, not aggressive

### EQ
```
HPF: 30-40 Hz
Body: 80-150 Hz
Cut: 200-400 Hz (mud)
Articulation: 700-1000 Hz (fingers)
```

### Compression
```
Ratio: 4:1
Attack: 20-40 ms
Release: 100-200 ms
GR: 4-6 dB
```

### Upright vs Electric
- **Upright:** More acoustic, traditional
- **Electric:** More common, P-bass tone typical

## Drum Production

### Blues Drum Sound

**Character:**
- Not modern and clicky
- Organic, vintage
- Shuffle groove
- Dynamic (loud/soft)

**Kick:**
```
EQ:
- HPF: 40-50 Hz
- Boost: 60-100 Hz (thump)
- Cut: 300-400 Hz (boxiness)
- Light boost: 2-4 kHz (attack, subtle)
Compression: Gentle, 3:1
```

**Snare:**
```
EQ:
- HPF: 80 Hz
- Boost: 150-200 Hz (body)
- Presence: 2-4 kHz (crack)
Allow some ring (not gated)
Brush playing: Very delicate EQ
```

**Hi-Hat/Cymbals:**
```
Natural, not hyped
HPF: 300-400 Hz
No excessive high boosts
```

### Shuffle Pattern
```
Kick: 1 . . 2 . . 3 . . 4 . .
Snare: . . . X . . . . . X . .
Hi-hat: x . x x . x x . x x . x (shuffle triplet)
```

## Harmonica Production

### Amped Harmonica Sound

**Classic Chain:**
```
Harmonica → Bullet mic → Small tube amp (overdriven) → Room mic
```

**EQ:**
```
HPF: 150-200 Hz
Cut: Excessive highs (3 kHz+)
Character: Midrange focused, gritty
```

**Character:** Raw, overdriven, not clean

## Hammond B3 / Organ

### Leslie Speaker

**The Sound:** Rotating speakers create doppler effect

**EQ:**
```
HPF: 80 Hz
Low rotor: 100-200 Hz (warmth)
High rotor: 2-4 kHz (shimmer, movement)
Don't remove: The Leslie character
```

### Settings
- **Fast/Slow toggle:** For expression
- **Drawbar settings:** Genre specific
- **Percussion:** On for attack

## Effects

### Tremolo

**Classic Blues Effect:**
```
Rate: Slow to medium
Depth: Moderate
Use: Guitar, sometimes organ
```

### Spring Reverb

**The Blues Reverb:**
```
Built into Fender amps
Decay: 1-2 seconds
Mix: To taste
Character: Splashy, distinctive
```

### Tube Saturation

**Essential for:**
- Warmth on everything
- Guitar tone shaping
- Vocal character
- Mix glue

## Mixing Blues

### Balance Goals

1. **Vocal/Guitar forward:** Stars of the show
2. **Rhythm section support:** Foundation, not dominant
3. **Warmth:** Analog character
4. **Dynamics:** Preserve the feel
5. **Space:** Intimate, like a club

### Typical Balance
```
Vocals: Front and center
Guitar: Prominent, supportive
Bass + Drums: Foundation
Keys: Fill, accent
Harp: Featured when playing
```

### Stereo Image
```
Center: Vocals, bass, kick, snare
L/R: Guitars (if multiple)
Slight spread: Drums, keys
Wide: Reverb returns
```

### Target Loudness

| Style | Target LUFS |
|-------|-------------|
| Traditional Blues | -14 to -16 |
| Blues Rock | -10 to -12 |
| Modern Blues | -12 to -14 |

## Subgenre-Specific Tips

### Delta Blues

- **Acoustic:** Guitar, voice
- **Production:** Minimal, raw
- **Character:** Sparse, emotional
- **Slide guitar:** Prominent
- **Recording:** Room sound, vintage

### Chicago Blues

- **Electric:** Full band sound
- **Harmonica:** Amped, gritty
- **Bass:** Walking lines
- **Feel:** Urban, powerful
- **Reference:** Chess Records sound

### Texas Blues

- **Guitar:** Loud, aggressive
- **SRV approach:** Clean amp cranked + Tube Screamer
- **Dynamics:** From whisper to scream
- **Feel:** Intense, virtuosic
- **Reference:** Stevie Ray Vaughan, Albert Collins

### British Blues

- **Sound:** More rock-influenced
- **Marshall amps:** Crunchier tone
- **Feel:** Heavier than American
- **Reference:** Early Clapton, Peter Green

### Blues Rock

- **Modern production:** More polished
- **Guitar tones:** Higher gain acceptable
- **Mix:** Louder, more compressed
- **Reference:** Joe Bonamassa, Gary Clark Jr.

## Common Mistakes

1. **Too clean:** Blues needs grit and character
2. **Over-compressed:** Kills dynamics and feel
3. **Too bright:** Blues is warm, not harsh
4. **Wrong reverb:** Too long or artificial
5. **Ignoring shuffle:** The groove is everything
6. **Over-polished vocals:** Raw is authentic
7. **Forgetting dynamics:** Soft to loud is essential

## Production Tips

1. **Learn the shuffle:** Feel is fundamental
2. **Leave space:** Blues breathes
3. **Embrace imperfection:** Perfection isn't the goal
4. **Use tube warmth:** Saturation everywhere
5. **Dynamics matter:** Pianissimo to fortissimo
6. **Call and response:** Vocal-guitar conversation
7. **Reference classics:** Stax, Chess, Blue Note recordings
8. **Room sound:** Ambience adds authenticity
