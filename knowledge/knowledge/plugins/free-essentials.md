# Free Plugin Essentials

## Overview

This guide lists high-quality free VST plugins that can achieve professional results. All plugins are legitimate freeware from reputable developers.

## Compressors

### TDR Kotelnikov
- **Type**: Mastering-grade transparent compressor
- **Best for**: Mix bus, mastering, transparent compression
- **Link**: tokyo-dawn.net
- **Features**: Clean, surgical, lookahead

### TDR Molotok
- **Type**: Character compressor
- **Best for**: Drums, parallel compression
- **Features**: Colorful, punchy

### Analog Obsession LALA
- **Type**: LA-2A style optical
- **Best for**: Vocals, bass, smooth compression
- **Link**: analogobsession.com

### Analog Obsession Fetish
- **Type**: 1176 style FET
- **Best for**: Vocals, drums, aggressive compression

### DC1A (Klanghelm)
- **Type**: Simple one-knob compressor
- **Best for**: Quick compression, beginners
- **Link**: klanghelm.com

### ReaComp (Reaper)
- **Type**: Versatile VCA-style
- **Best for**: Everything, comes with Reaper
- **Features**: Sidechain, flexible routing

## Download Sources

**Safe, legitimate sources:**
- Developer websites (linked above)
- Plugin Boutique (curated free section)
- KVR Audio (database of free plugins)
- Bedroom Producers Blog (reviews and links)

**Avoid:**
- Torrent sites (malware risk, illegal)
- Crack sites (malware, viruses)
- Unknown third-party sites

## Installation Tips

1. Always download from official sources
2. Scan with antivirus before installing
3. Install to standard VST folder
4. Rescan plugins in Reaper after installing
5. Keep installers backed up

## Reaper's Built-in Plugins

Don't forget Reaper includes excellent plugins:
- ReaEQ, ReaComp, ReaDelay, ReaVerb
- ReaGate, ReaXcomp, ReaLimit, ReaFIR
- JS plugin collection (hundreds of effects)

These are professional quality and already installed!
